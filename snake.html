Snake Game

This was my first medium level project

I created this game using HTML, CSS, and JS. This was the first game I have ever created. When I was younger I used to play this game on my moms old nokia phone, and when I was younger I always wanted to recreate it. The main element of design used was Canvas, this was also my first time working with canvas which I found out to be super fun. Also learned to create many function in JS such as, startGame(x), with parameters, pauseGame(), endGame(). These were the functions running the whole game.
