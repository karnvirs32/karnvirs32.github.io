Checkers Game

HTML CSS JS

Since I was younger I loved checkers, or table games in general. It was one of those games I used to play with my family on gatherings or parties. This was the first game I created where I adapted JS to create a smart game. This game took be about a month to create just to come up with a small algorithm, or game loop, where the computer learned from the moves of the player. I had to google some checker techniques used from other developers to come up with a working game loop.
