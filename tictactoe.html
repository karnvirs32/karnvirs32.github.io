Tic Tac Tow

I created this game using HTML CSS and JS

I created this game simply using HTML, CSS, and JS. This one of the project that I created the designing of the elements without the help of canvas. Simply used CSS to design the tiles by giving them borders. The first section has our title within a h1. The second section displays whose turn it is currently. Inside the display we have a span which will contain X or O depending on the current user. I applyed classes to a span to colorize the text. The third section is the one that holds the board for the game. It has a container class so it can properly place the tiles. Inside this section I have nine divs which will act as the tiles inside the board.
